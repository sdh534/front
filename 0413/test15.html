    0         1         2         3
    0123456789012345678901234567890
    ABCDEFGHIJKLMNOPQRSTUVWXYZabcde
    ABC   BCDEF   GHI   JKLM......  ABC.... OPQ

length: 객체(문자(열))의 길이

substring(a,b): 문자열 발췌(a에서부터 b 앞까지)

slice(a,b): 문자열 발췌(a에서부터 b 앞까지)

substr()

replace(a,b): a를 b로 치환

replaceAll(a,b): a를 b로 모두 치환

toUpperCase() : 대문자로 변환

toLowerCase() : 대문자로 변환

concat(a,b) : a와 b 문자열 결합

trim(): 문자열 앞뒤(양쪽) 공백무시

trimStart(): 문자열의 앞쪽 공백 무시

trimEnd(): 문자열의 앞쪽 공백 무시

replaceAll(" ","") : 공백 모두 제거

charAt : 하나의 문자만 발췌

charCodeAt(a): a문자의 아스키코드를 출력

split(): 특정 문자를 기준으로 잘라냄. 이때 잘라낸 값들은 배열변수에 기억됨

indexOf(): 문자(열) 검색시에 사용 : 있으면 위치값을, 없으면 -1

lastIndexOf(): 문자(열) 검색시에 사용(뒤에서부터 검색) : 있으면 위치값을, 없으면 -1을 반환

search() : 문자(열) 검색시에 사용: 있으면 위치값을, 없으면 -1을 반환

match() : 문자열 일치 여부... 있으면 '문자열 그대로'반환, 없으면 공백반환

includes() : 문자열 포함 여부... 있으면 true, 없으면 false
